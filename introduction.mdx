title: Introdução
---

Está documentação irá abordar as nuances da api utilizada na plataforma atlas da speedio, mais especificamente a api em Ruby
On Rails.

Seus propósitos principais são:
#### Consulta de dados
Ela contém endpoints usados para consulta de dados de empresas, seja buscando via cnpj ou buscas mais abrangentes visando
a quantidade de empresas baseadas em filtragens.
#### Gestão de assinaturas
Faz uso de uma integração com o ERP de pagamento superlogica para cuidar do gerenciamento de assinaturas, pagamentos e distribuição
de creditos.
#### Autenticação
Operações de autenticação como criação de conta, login, recuperação de senha.
